# 在线打赏系统

向导式安装，一个脚本搞定「打赏」系统。[演示站点](https://pay.biezhi.me)

> 部分功能正在开发，核心功能已完备。
> 有任何问题请发 [issue](https://github.com/biezhi/profit/issues/new)，注意逻辑清晰！

## 特性

- 支持修改主题模板
- 支持打赏主题扩展
- 支持支付渠道扩展
- 使用 SQLite，无需安装数据库
- 部署简单，支持脚本运行
